【服务器/电脑】环境搭建

  • Mac电脑上下载Cyberduck，用于和服务器文件传输

  • Mac上官网下载Anaconda3，传到服务器上，bash Anaconda3-5.0.1-Linux-x86_64.sh 根据提示进行安装。（确认安装位置，YES同意条款等）

  • 输入conda --version 如果提示conda命令没找到，那就需要配置环境变量

  • 输入 conda --version 如果出现版本就证明 anaconda 安装成功，接下来创建自己的环境。

  • anaconda 各个版本的下载

  • 清华镜像下载各种包

  • 安装tensorflow

  • 安装opencv

  • 版本对应关系：tensorflow –> python 3.5 –> Anaconda3-4.2.0

    Anaconda3-4.2.0-Linux-x86_64.sh

  • 配置cuda环境变量

  • hello world!
